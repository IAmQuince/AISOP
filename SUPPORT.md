# Support

For local zip handoff, include the package version, revision, generated audit reports, and the exact command that failed.

For GitHub use, open an issue using the closest issue template. Include:

- package or repository version;
- branch name;
- operating system;
- Python version;
- audit report excerpts;
- expected behavior;
- actual behavior.

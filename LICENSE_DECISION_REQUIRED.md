# License Decision Required

This seed package intentionally does not choose a license for the user.

Before publishing this package or a derived project as open source, choose and add an appropriate `LICENSE` file. Until then, treat the package as **not licensed for public reuse** except where the owner separately grants permission.

## Decision checklist

- Is the project intended to be open source, internal, commercial, dual-licensed, or proprietary?
- Are third-party snippets, templates, or examples included?
- Are hardware-control or safety-related materials included?
- Does the chosen license match the intended reuse model?
- Has attribution been added to `NOTICE.md`?
